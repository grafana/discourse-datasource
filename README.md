Create an API key in discourse...

## Getting started

1. Install dependencies

```BASH
yarn install --pure-lockfile
```

2. Build plugin in development mode or run in watch mode

```BASH
yarn dev
```

or

```BASH
yarn watch
```

3. Build plugin in production mode

```BASH
yarn build
```
